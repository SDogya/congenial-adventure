# Журнал взаимодействия с ИИ-ассистентами

В данной директории собраны (почти) все сырые логи и системные промпты, использованные в ходе работы. Для удобства проверки логи разделены на следующие этапы:

* **`/papers/`** — Логи использования RAG-ассистентов (DeepResearch, NotebookLM, Gemini) для поиска литературы, извлечения данных из статей (BLT, OAT, H-Net) и генерации обзора.
* **`/roles/`** — Системные промпты и ролевые инструкции, которые использовались для настройки агентов (Архитектор, Ревьюер, Кодер и т.д.) перед генерацией кода.
* **`/NEURO_LOG/`** — Основные транскрипты бесед и логи терминала при работе с Claude Code.
  * *`HYP_TO_CODE.md`* — Процесс трансформации математической гипотезы FD-DRAT в структуру PyTorch проекта.
  * *`coder_log.md` / `auditor_log.md`* — Логи написания и статического аудита кода.
  * *`фиксим размерности.md`* — Логи сессий дебага c perplexity (токены закочились) (борьба с tensor mismatch и OOM на Kaggle).
