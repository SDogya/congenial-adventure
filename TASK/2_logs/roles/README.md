Роли ИИ-агентов, разработанные промпт-инженером на основе обзоров статей и загруженные в NotebookLM.

Идея состояла в создании конвейера специализированных агентов:

1. Составлено несколько обзоров по каждой статье и по направлению VLA в целом.
2. Материалы переданы в NotebookLM; на их основе запущен цикл «гипотезёр — критик».
3. После одобрения гипотеза передана промежуточному агенту между гипотезёром и кодером: декомпозиция задачи снижает вероятность ошибок.
4. Кодер работал в паре с ревьювером.
5. Роль «дизайнера эксперимента» была предусмотрена, но полноценно не задействована.

Не все роли из этой папки были применены на практике.
